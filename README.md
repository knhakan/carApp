# CarApp demo application

A simple repository to implement login and switch the language based on need.  
The project contains both Backend written in NodeJs and frontend written in ReactJs.


## Getting started

To install the project, you need to run the following commands both for backend and for frontend  

Install packages:    
npm -> run the command: npm install  
yarn -> run the command: yarn  

Run:  
npm -> run the command: npm start  
yarn -> run the command: yarn start  


### APIs  

*/auth/login*  

credentials for login can be changed in db.json file. The default values are:  
{  
    "username" : "admin",  
    "password" : "123123"  
},  
{  
    "username": "admin@gmail.com",  
    "password": "123123"  
}
